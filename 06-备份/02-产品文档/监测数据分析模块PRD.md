## 1. 业务背景与使用角色

### 1.1 业务背景
- 模块定位：监测数据分析模块基于测试示范全量数据，提供测试数据管理、查询检索、事故分析研判及三大专题（道路测试、示范应用、示范运营）的可视化分析功能
- 核心特点：以数据可视化驱动决策，所有分析图表均支持时间范围选择、维度筛选、图形导出及数据下钻
- 关联模块关系：
  - 测试示范监管模块：引用企业接入信息、车辆信息、安全员信息作为分析数据源
  - 平台管理模块：引用字典配置、日志记录等基础数据
  - 车辆监控模块：实时位置、告警数据、运行状态等通过消息队列推送至分析模块

### 1.2 使用角色

| 角色 | 该模块中的职责 | 客户端 |
| ---- | ------------ | ------ |
| 政府监管人员 | 查看全局数据分析、事故研判、产业趋势 | Web端 |
| 企业用户 | 查看本企业相关数据分析和查询结果 | Web端 |
| 系统管理员 | 管理数据权限、查看审计日志 | Web端 |

### 1.3 核心目标与价值

| 目标维度 | 说明 |
| -------- | ---- |
| 数据总览 | 一站式展示测试企业、车辆、里程、告警等关键指标 |
| 异常监测 | 实时监测告警状态、处理进度和趋势变化 |
| 事故分析 | 从解析、成因、特征、研判四个维度深度分析事故数据 |
| 产业洞察 | 分析企业增长、测试规模、场景渗透等产业发展指标 |
| 专题分析 | 按道路测试、示范应用、示范运营三个专题分维度分析 |
| 数据查询 | 支持企业、车辆、安全员、设备、故障等多维数据查询与导出 |

## 3. 功能详细设计

### 3.1 F01 测试数据总览

#### 3.1.1 页面布局
页面自上而下分为6个区域：
1. **时间筛选栏**：快捷时间按钮 + 自定义起止日期选择器 + 查询/刷新按钮
2. **累计统计卡片区**：单行4个卡片（测试企业数/临牌发放数/测试时长/下发提醒）
3. **图表区**：两列布局（测试主体累计测试里程横向柱状图 + 测试里程趋势分析组合图）
4. **当日测试情况区**：6个紧凑卡片 + 手动刷新按钮
5. **当前测试企业车辆统计表**：企业名称/当日测试车辆数/正在测试车辆数
6. **实时车辆动态列表**：5条/页分页列表 + 8秒自动轮换刷新

#### 3.1.2 查询筛选区
| 元素 | 类型 | 配置说明 |
| ---- | ---- | -------- |
| 快捷时间 | 按钮组 | 枚举值：今日、昨日、最近7天（默认选中）、最近30天、本月 |
| 开始时间 | 日期选择器 | 联动快捷时间自动填充 |
| 结束时间 | 日期选择器 | 联动快捷时间自动填充 |
| 查询按钮 | 主按钮 | 触发全部卡片和图表数据刷新 |
| 刷新按钮 | 次按钮 | 刷新累计数据，带旋转图标 |

**查询条件显示规则：** 筛选条件一行展示，下方辅助文字说明"以下累计数据和图表共用时间筛选"。

#### 3.1.3 图表区
| 图表名称 | 图表类型 | 数据维度 | 说明 |
| ---- | ---- | ---- | ---- |
| 测试主体累计测试里程 | 横向柱状图 | X轴为里程(km)，Y轴为企业名称 | 所选周期内各企业累计里程，柱体右侧显示数值 |
| 测试里程趋势分析 | 组合图（折线+面积+虚线） | 自动驾驶里程（绿色面积折线）、人工驾驶里程（橙色折线）、自动驾驶占比（蓝色虚线，副坐标轴） | 时间粒度按筛选范围适配：今日/昨日按时段，最近7天按日，最近30天/本月按周 |

#### 3.1.4 当日测试情况卡片
| 卡片标题 | 说明 |
| ---- | ---- |
| 当前测试企业数 | 今日有测试活动的企业数 |
| 测试车辆数 | 今日参与测试车辆总数 |
| 当日测试里程(km) | 今日累计测试里程 |
| 当日测试时长(小时) | 今日累计测试时长 |
| 载人测试总里程(km) | 载人测试累计里程 |
| 载人测试总时长(小时) | 载人测试累计时长 |

- 带手动刷新按钮（图标+点击触发），点击可模拟实时数据波动

#### 3.1.5 数据列表区

**当前测试企业车辆统计表：**
| 序号 | 字段名 | 字段标识 | 宽度建议 | 对齐方式 | 说明 |
| ---- | ------ | -------- | -------- | -------- | ---- |
| 1 | 企业名称 | company_name | auto | 左 | |
| 2 | 当日测试车辆数 | test_vehicles | 150px | 中 | 该企业当日测试车辆总数 |
| 3 | 正在测试车辆数 | active_vehicles | 150px | 中 | 自动驾驶/人工驾驶且车速>0的车辆数，绿色标签展示 |

**实时车辆动态列表：**
| 序号 | 字段名 | 字段标识 | 宽度建议 | 对齐方式 | 说明 |
| ---- | ------ | -------- | -------- | -------- | ---- |
| 1 | 临时号牌 | plate | 140px | 左 | 蓝色高亮显示 |
| 2 | 所属企业 | enterprise | 120px | 左 | |
| 3 | 驾驶模式 | mode | 100px | 中 | 自动驾驶（绿色标签）、人工驾驶（黄色标签）、远程驾驶（蓝色标签） |
| 4 | 车速(km/h) | speed | 80px | 中 | |
| 5 | 最后上报时间 | time | 160px | 中 | |

**排序规则：** 按最后上报时间倒序排列。
**分页规则：** 默认每页5条，见附录A.1（特殊场景：每页5条）。

**特殊设计：** 8秒自动轮换刷新机制，模拟新上报数据，带数据总数显示。

#### 3.1.6 交互流程
1. 页面加载 → 默认选中"最近7天" → 渲染全部卡片、图表、表格、实时列表
2. 用户点击快捷时间按钮 → 更新筛选日期 + 刷新累计数据和图表 + 按钮高亮切换
3. 用户自定义日期后点击"查询" → 刷新全部数据 + Toast提示"数据已更新"
4. 用户点击"刷新" → 刷新累计数据 + Toast提示"累计数据已刷新"
5. 用户点击当日测试情况刷新图标 → 模拟数据波动 + 刷新当日卡片和统计表 + Toast提示"实时数据已刷新"
6. 实时车辆动态列表每8秒自动刷新 → 更新上报时间和车速（模拟数据轮换）

## 4. 业务规则汇总

### 4.1 计算规则
| 规则名称 | 计算公式 | 说明 |
| ---- | ---- | ---- |
| 脱手率 | (脱离次数 / 自动驾驶里程) × 100 | 分母为0时显示"-" |
| 干预率 | (干预次数 / 自动驾驶里程) × 100 | 分母为0时显示"-" |
| 数据完整率 | (实际上报条数 / 应上报条数) × 100% | |
| 告警处理率 | (已处理告警数 / 告警总数) × 100% | |
| 严重告警占比 | (严重告警数 / 告警总数) × 100% | |
| 预警→事故转化率 | (关联事故数 / 预警事件总数) × 100% | |
| 自动驾驶占比 | (自动驾驶里程 / 总里程) × 100% | 百分比取1位小数 |

### 4.2 编号生成规则
| 规则项 | 告警编号 | 事故编号 | 预警事件编号 |
| ---- | ---- | ---- | ---- |
| 前缀 | ALT | ACC | EVT |
| 日期部分 | YYYYMMDD（取自时间字段） | YYYY | YYYY |
| 序号部分 | 4位流水号 | 4位流水号 | 5位流水号 |
| 示例 | ALT-20260509-0001 | ACC-2026-0001 | EVT-2026-00001 |

### 4.3 时间粒度适配规则
| 筛选范围 | 图表时间粒度 |
| ---- | ---- |
| 今日/昨日 | 按时段（每4-6小时） |
| 最近7天 | 按日 |
| 最近30天 | 按周 |
| 本月 | 按周 |

### 4.4 导出规则
| 规则项 | 说明 |
| ---- | ---- |
| 导出格式 | Excel(.xlsx) |
| 导出上限 | 当前筛选条件下全量结果（上限10万条），超限提示缩小范围 |
| 文件命名 | {模块名称}_{导出日期}.xlsx |

### 4.5 权限规则
| 角色 | 数据可见范围 |
| ---- | ---- |
| 政府监管人员 | 全局数据 |
| 企业用户 | 仅本企业相关数据 |
| 系统管理员 | 全局数据 + 审计日志 |

### 4.6 数据刷新规则
| 页面 | 刷新机制 |

## 附录A：统一组件规范

> 以下组件在本系统多个模块中复用，功能详细设计中引用这些组件时仅需注明"见附录A.x"。

### A.1 分页组件
| 配置项 | 规则 |
| ---- | ---- |
| 每页条数选项 | 10、20、50、100，默认10 |
| 显示信息 | 总记录数、当前页码/总页数 |
| 翻页控件 | 首页、上一页、页码（带省略号省略）、下一页、末页 |
| 切换每页条数 | 下拉选择，切换后从第一页重新加载 |

**特殊场景分页配置：**
| 配置项 | 规则 |
| ---- | ---- |
| 实时车辆动态列表 | 默认5条/页 |
| 查询类列表（企业/安全员/车辆/设备/故障） | 默认20条/页 |

### A.2 删除确认组件
本模块不涉及删除功能。

### A.3 导出组件
| 触发条件 | 行为 |
| ---- | ---- |
| 用户选中了特定数据 | 导出用户选择的数据，不支持跨页导出 |
| 用户未选择特定数据 | 导出当前筛选条件下的全量数据 |

**导出格式：** .xlsx
**文件命名规则：** {功能名称}_{YYYYMMDD}.xlsx
**导出上限：** 10万条，超限提示"数据量过大，请缩小筛选范围后重试"

### A.4 附件上传组件
| 规则项 | 图片类 | 文档类 |
| ---- | ------ | ------ |
| 支持格式 | jpg/png | pdf |
| 单文件大小限制 | ≤5MB | ≤20MB |
| 多文件上传 | 支持（最多5张） | 支持 |
| 预览/下载 | 查看态支持点击预览/下载 | 查看态支持点击下载 |

### A.5 复选框组件
- 表头存在全选复选框，勾选后选中当前页所有行
- 支持单行勾选
- 全选仅作用于当前页数据，不支持跨页全选

### A.6 查询筛选区组件
**通用布局规则：**
- 查询条件与查询按钮、重置按钮在同一行排列
- 查询按钮为主按钮（蓝色），重置按钮为次按钮
- 导出按钮位于筛选行右侧

### A.7 二次确认弹窗组件
**通用规则：**
- 弹窗宽度：400~500px
- 位置：居中弹出
- 遮罩：半透明，点击遮罩可关闭
- 确定按钮：主按钮
- 取消按钮：次按钮
